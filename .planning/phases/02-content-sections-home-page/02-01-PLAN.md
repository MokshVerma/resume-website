**Metric highlighting utility** (define within this file or as a separate function at the top):
```typescript
function highlightMetrics(text: string): React.ReactNode {
  const metricPattern = /([~]?\$?\d[\d,.]*[MKBmkb]?\+?(?:\s*(?:DAU|%)))/g;
  const parts = text.split(metricPattern);
  if (parts.length === 1) return text;
  return parts.map((part, i) =>
    i % 2 === 1 ? (
      <span key={i} className="text-accent font-bold">{part}</span>
    ) : (
      <span key={i}>{part}</span>
    )
  );
}
```
This regex matches: 380M+, $200K+, 50M DAU, ~30%, 50%, 10%, 40%, 35% -- all metrics in the data. Uses split with capture group so odd indices are matches (per RESEARCH.md Pattern 3). Do NOT use `.test()` in a loop with a global regex (RESEARCH.md Pitfall 2).
