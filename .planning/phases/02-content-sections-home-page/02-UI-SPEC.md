# Phase 2 — UI Design Contract

> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

| md | 16px (4 units) | `p-4`, `gap-4` | Card internal padding (mobile), element gaps |
| lg | 24px (6 units) | `p-6`, `gap-6` | Card internal padding (desktop), element groups |

### Section Spacing Contract (Phase 2 specific)

Each content section is separated by consistent vertical padding. Mobile uses tighter spacing; desktop uses looser spacing.

| Section | Mobile (`py-`) | Desktop (`md:py-`) | Notes |
|---------|---------------|---------------------|-------|
| Hero | `py-24` (96px) | `md:py-32` (128px) | Full viewport height via `min-h-screen` with flex centering |
| Experience | `py-12` (48px) | `md:py-16` (64px) | Longest section, standard spacing |
| Skills | `py-12` (48px) | `md:py-16` (64px) | Standard spacing |
| Projects | `py-12` (48px) | `md:py-16` (64px) | Standard spacing |
| Education | `py-12` (48px) | `md:py-16` (64px) | Minimal section, same rhythm |
| Footer | `py-8` (32px) | `md:py-12` (48px) | Reduced -- footer is subordinate |

## Color

Inherited from Phase 1 `globals.css`. No new tokens needed for Phase 2.

### Palette

| Token | Hex | Role | Usage |
|-------|-----|------|-------|
| `--color-background` | `#0a0a0a` | Dominant (60%) | Page background |
| `--color-surface` | `#141414` | Secondary (30%) | Experience cards, project cards, footer background |
| `--color-border` | `#27272a` | Structural | Card borders, section dividers |
| `--color-foreground` | `#ededed` | Primary text | Name, headings, primary content |
| `--color-muted` | `#a1a1aa` | Secondary text | Descriptions, metadata, dates, locations |
| `--color-accent` | `#3b82f6` | Accent (10%) | See reserved-for list below |
| `--color-accent-hover` | `#60a5fa` | Accent hover | Hover state for accent elements |
| `--color-accent-muted` | `#3b82f615` | Accent background | Skill chip backgrounds, subtle highlights |

### Accent Reserved For (Phase 2 elements)

The accent color `#3b82f6` is used for these Phase 2 elements ONLY:

1. **Primary CTA button background** -- "Download Resume" in hero
2. **Impact metric numbers** -- 380M+, $200K+, 50M DAU, ~30%, etc. in experience bullets
3. **Job title text** -- role names in experience cards for quick scanning
4. **Tech stack chip text** -- in project cards, `font-mono` treatment
5. **GitHub link icon/text** -- on project cards, hover state
6. **Timeline accent dot** -- the marker connecting experience cards

Accent is NOT used for: section headings, card borders, all hover states generically, skill chip text, education text, footer links, or decorative purposes beyond the existing background gradient.

| Property | Value | CSS |
|----------|-------|-----|

| Border | `1px solid rgba(39, 39, 42, 0.5)` | Semi-transparent border |
| Hover background | `rgba(20, 20, 20, 0.8)` | Slightly more opaque on hover |
| Hover border | `1px solid var(--color-accent)` | Accent border on hover |

### Component Specifications

#### HeroSection

- Layout: `min-h-screen flex flex-col items-center justify-center text-center`
- Greeting: dynamically computed from client local time (requires `"use client"`)
- Greeting format: "Good morning", "Good afternoon", or "Good evening" (no comma, no name)
- Greeting thresholds: morning 5-11, afternoon 12-16, evening 17-4
- Name: `h1` with fluid clamp sizing
- Title: `p` styled as heading size in muted color
- Summary: `p` in body size, muted color, `max-w-2xl` for comfortable line length
- CTAs: flex row on `sm:` and up, stacked column below
- "Download Resume" CTA: `bg-accent text-background font-bold rounded-lg px-6 py-3` with `hover:bg-accent-hover`
- "Get in Touch" CTA: `border border-border text-foreground font-bold rounded-lg px-6 py-3` with `hover:border-accent hover:text-accent`
- Horizontal page padding: `px-6 md:px-12 lg:px-24`
- Content max width: `max-w-4xl mx-auto`
- Section `id="hero"` for future nav linking (Phase 3)

#### ExperienceSection

- Section `id="experience"` for future nav linking
- Heading: "Experience" via SectionHeading
- Layout: vertical stack of ExperienceCards with `gap-6`
- Timeline visual: a thin vertical line (`2px wide, bg-border`) running along the left edge of cards, with accent dots at each card's top
- Timeline dot: `w-3 h-3 rounded-full bg-accent` absolutely positioned on the line
- On mobile: no timeline line, just cards stacked with dot markers
- On desktop: timeline line visible, cards offset to the right with `pl-8 md:pl-12` relative to timeline

#### ExperienceCard

- Container: `bg-surface rounded-xl border border-border p-4 md:p-6`
- Border radius: `12px` (Tailwind `rounded-xl`)
- Header row: company name (bold, foreground) and date/location (small, muted) on opposite ends (flex justify-between), wrapping on mobile
- Role title: body size, accent color, below company name
- Description (if present): body size, muted, below role title
- Bullets: unordered list with `space-y-2`
- Bullet marker: custom accent-colored dash or circle, not default disc
- Impact metrics regex: numbers followed by M+, K+, DAU, %, or dollar amounts -- wrap in `<span className="text-accent font-bold">`
- Impact metric detection pattern: `/(\d[\d,.]*[+]?)\s*(M\+|K\+|DAU|%|x)/g` and `/\$[\d,.]+[+K]*/g`

#### SkillsSection

- Section `id="skills"` for future nav linking
- Heading: "Skills" via SectionHeading
- Layout: grid of skill categories, `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6`
- Each category: a group with category label on top
- Category label: small size, muted, `font-bold uppercase tracking-wider mb-3`
- Chips container: `flex flex-wrap gap-2`
- Individual chip: `px-3 py-1 rounded-full bg-accent-muted text-foreground text-sm border border-border`
- Chip hover: none (chips are not interactive in Phase 2)

#### ProjectsSection

- Section `id="projects"` for future nav linking
- Heading: "Projects" via SectionHeading
- Layout: `grid grid-cols-1 md:grid-cols-2 gap-6`
- Each project rendered as a ProjectCard

#### ProjectCard (Glassmorphism)

- Container: `rounded-xl p-4 md:p-6` with glassmorphism styles (see Color section)
- CSS: `background: rgba(20, 20, 20, 0.6); backdrop-filter: blur(12px); border: 1px solid rgba(39, 39, 42, 0.5);`
- Project name: heading size, bold, foreground
- Description: body size, muted, `mt-2`
- Tech stack: `flex flex-wrap gap-2 mt-4`
- Tech chip: `px-2 py-1 rounded-md bg-accent-muted text-accent text-sm font-mono`
- GitHub link: `mt-4 inline-flex items-center gap-2 text-sm text-muted hover:text-accent transition-colors`
- GitHub icon: `FiGithub` from react-icons/fi, 16px
- External link icon: `FiExternalLink` from react-icons/fi, 14px, next to GitHub text
- **Hover effect:** `transition-all duration-300 hover:-translate-y-1 hover:shadow-[0_8px_32px_rgba(59,130,246,0.15)]` -- subtle lift with accent glow shadow
- Hover border: `hover:border-accent/50` (accent at 50% opacity)

#### EducationSection

- Section `id="education"` for future nav linking
- Heading: "Education" via SectionHeading
- Minimal layout: no card wrapper
- Degree: body size, bold, foreground
- Institution: body size, muted
- Location and period: small size, muted, on same line separated by a bullet character
- No icons, no decorative elements

#### Footer

- Background: `bg-surface`
- Padding: `py-8 md:py-12 px-6 md:px-12 lg:px-24`
- Content max width: `max-w-4xl mx-auto`
- Layout: centered, stacked
- Social links row: `flex items-center justify-center gap-6`
- GitHub icon: `FiGithub` from react-icons/fi, 20px
- LinkedIn icon: `FiLinkedin` from react-icons/fi, 20px
- Email icon: `FiMail` from react-icons/fi, 20px
- Icon styling: `text-muted hover:text-foreground transition-colors`
- Resume download link: below social icons, `mt-4 text-sm text-muted hover:text-accent transition-colors underline`
- Copyright: below resume link, `mt-4 text-sm text-muted` -- "Moksh Verma" (current year auto-generated is out of scope; use static text if needed)
- All links open external URLs in new tab (`target="_blank" rel="noopener noreferrer"`) except email (mailto:) and resume PDF (same tab download)

#### SectionHeading (Shared)

- Props: `title: string`, `id?: string`
- Render: `<h2 className="text-heading font-bold leading-heading text-foreground mb-8 md:mb-12">{title}</h2>`
- Used by: ExperienceSection, SkillsSection, ProjectsSection, EducationSection

#### MetricHighlight (Shared)

- Purpose: parses experience highlight strings and wraps impact metrics in accent spans
- Props: `text: string`
- Logic: regex match numbers + suffixes (M+, K+, DAU, %, dollar amounts), wrap each match in `<span className="text-accent font-bold">`
- Render: returns JSX with mixed text and highlighted spans
- Used by: ExperienceCard for each bullet point

| Section heading: Projects | "Projects" | Hardcoded structural label |
| Section heading: Education | "Education" | Hardcoded structural label |
| GitHub link text on project cards | "View on GitHub" | Hardcoded label |
| Resume download (footer) | "Download Resume (PDF)" | Hardcoded label |

### Content Source Rule

All resume content (name, title, summary, experiences, skills, projects, education, social links) must be imported from `@/lib/data`. No hardcoded resume content in components. Only structural labels (section headings, CTA text, link labels) are hardcoded.

### Heading Hierarchy

| Level | Content | Component |
|-------|---------|-----------|

### Landmark Structure

| Element | HTML | Phase 2 Changes |

| Section | ID | Element |

### Accessibility

- All social link `<a>` tags must have `aria-label` (e.g., `aria-label="GitHub profile"`, `aria-label="LinkedIn profile"`, `aria-label="Send email"`)
- Resume download link: `aria-label="Download resume as PDF"`
- Project GitHub links: `aria-label="View {project name} on GitHub"`
- External links: `target="_blank" rel="noopener noreferrer"`
- Skill chips: not interactive, no role needed (decorative grouping)
- Impact metric highlights: no special ARIA needed (visual emphasis only, text content is unchanged)

## Interaction States

Phase 2 has limited interactivity (no navigation, no forms). Interactions are hover/focus states only.

| Element | Default | Hover | Focus | Active |
|---------|---------|-------|-------|--------|
| Primary CTA ("Download Resume") | `bg-accent text-background` | `bg-accent-hover` | `ring-2 ring-accent ring-offset-2 ring-offset-background` | Same as hover |
| Secondary CTA ("Get in Touch") | `border-border text-foreground` | `border-accent text-accent` | `ring-2 ring-accent ring-offset-2 ring-offset-background` | Same as hover |
| Project card | Glass background, border-border | `-translate-y-1`, accent glow shadow, `border-accent/50` | Entire card focus ring | Same as hover |
| GitHub link (project) | `text-muted` | `text-accent` | `underline` | Same as hover |
| Social icon (footer) | `text-muted` | `text-foreground` | `ring-2 ring-accent rounded` | Same as hover |
| Resume link (footer) | `text-muted underline` | `text-accent` | `ring-2 ring-accent rounded` | Same as hover |

All transitions: `transition-colors duration-200` for color changes, `transition-all duration-300` for transforms (project card lift).
