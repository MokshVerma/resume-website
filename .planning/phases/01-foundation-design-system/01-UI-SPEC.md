> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

| Property | Value |

## Spacing Scale

Declared values (must be multiples of 4). Tailwind v4 uses `--spacing: 0.25rem` as base unit, so all utilities (`p-1` = 4px, `p-2` = 8px, etc.) derive from this automatically.

| Token | Value | Tailwind Utility | Usage |
|-------|-------|-----------------|-------|
| xs | 4px (1 unit) | `p-1`, `gap-1` | Icon gaps, inline padding |
| sm | 8px (2 units) | `p-2`, `gap-2` | Compact element spacing, chip padding |
| md | 16px (4 units) | `p-4`, `gap-4` | Default element spacing, card padding |
| lg | 24px (6 units) | `p-6`, `gap-6` | Section padding on mobile, element groups |
| xl | 32px (8 units) | `p-8`, `gap-8` | Layout gaps, card padding on desktop |
| 2xl | 48px (12 units) | `p-12`, `gap-12` | Major section vertical padding (mobile) |
| 3xl | 64px (16 units) | `p-16`, `gap-16` | Major section vertical padding (desktop) |
| 4xl | 96px (24 units) | `p-24`, `py-24` | Page-level hero/section vertical spacing |

Exceptions: none

**Source:** Claude's Discretion (CONTEXT.md). Tailwind v4's default `--spacing: 0.25rem` base unit provides the 4px grid natively.

## Typography

Font: **Geist** (variable font, all weights 100-900 in single file). Loaded via `next/font/google` with `variable: "--font-geist-sans"`. Mapped to `--font-sans` in `@theme inline`.

Mono font: **Geist Mono** for code snippets and tech stack labels. Loaded via `next/font/google` with `variable: "--font-geist-mono"`. Mapped to `--font-mono` in `@theme inline`.

### Type Scale

| Role | Size | Rem | Weight | Line Height | Tailwind Class |
|------|------|-----|--------|-------------|----------------|
| Display | 72px | 4.5rem | 700 (bold) | 1.1 | `text-display font-bold leading-display` |
| H1 | 48px | 3rem | 700 (bold) | 1.2 | `text-h1 font-bold leading-heading` |
| H2 | 32px | 2rem | 600 (semibold) | 1.2 | `text-h2 font-semibold leading-heading` |
| H3 | 24px | 1.5rem | 600 (semibold) | 1.3 | `text-h3 font-semibold` |
| Body | 16px | 1rem | 400 (regular) | 1.6 | `text-body leading-body` |
| Small | 14px | 0.875rem | 400 (regular) | 1.5 | `text-sm` |
| XS | 12px | 0.75rem | 400 (regular) | 1.5 | `text-xs` |

### Weights Used

Only two weights to maintain visual hierarchy without clutter:

| Weight | Value | Usage |
|--------|-------|-------|
| Regular | 400 | Body text, descriptions, secondary content |
| Semibold | 600 | H2, H3 headings, labels, navigation items |
| Bold | 700 | Display text, H1, CTA buttons, impact metrics |

**Note:** Three weights listed but bold and semibold serve distinct roles (display/hero vs section headings). The visual hierarchy is: bold for "look here first," semibold for "this is a section," regular for "read this."

### Responsive Type Behavior

| Role | Mobile (< 768px) | Desktop (>= 768px) |
|------|-------------------|---------------------|
| Display | 2.5rem (40px) | 4.5rem (72px) |
| H1 | 2rem (32px) | 3rem (48px) |
| H2 | 1.5rem (24px) | 2rem (32px) |
| Body | 1rem (16px) | 1rem (16px) |

**Source:** D-05, D-06 (CONTEXT.md) for direction. RESEARCH.md for specific values. Responsive scaling is Claude's Discretion.

## Color

### Palette

| Token | Hex | Role | Usage |
|-------|-----|------|-------|
| `--color-background` | `#0a0a0a` | Dominant (60%) | Page background, body |
| `--color-surface` | `#141414` | Secondary (30%) | Card backgrounds, elevated sections, footer background |
| `--color-border` | `#27272a` | Structural | Dividers, card borders, input borders |
| `--color-foreground` | `#ededed` | Primary text | Headings, body text, primary content |
| `--color-muted` | `#a1a1aa` | Secondary text | Descriptions, metadata, placeholder text |
| `--color-accent` | `#3b82f6` | Accent (10%) | See reserved-for list below |
| `--color-accent-hover` | `#60a5fa` | Accent hover | Hover state for accent elements |
| `--color-accent-muted` | `#3b82f615` | Accent background | Subtle tag/chip backgrounds, hover highlights |

### 60/30/10 Distribution

| Split | Color | Where |
|-------|-------|-------|
| 60% Dominant | `#0a0a0a` (background) | `<body>`, page background, empty space |
| 30% Secondary | `#141414` (surface) + `#27272a` (border) | Cards, section backgrounds, timeline track, dividers, footer |
| 10% Accent | `#3b82f6` (accent) | Restricted to list below |

### Accent Reserved For (explicit list)

The accent color `#3b82f6` is reserved for these elements ONLY:

1. **Primary CTA button background** ("Download Resume" button)
2. **Impact metric numbers** in experience section (380M+, $200K+, 50M DAU)
3. **Active navigation indicator** (Phase 3)
4. **Link hover states** (text links on hover)
5. **Timeline accent marker** (dot or line highlight in experience timeline)
6. **Form submit button** (Phase 4)
7. **Background gradient glow** (subtle, at 0.08 opacity -- decorative, not interactive)

Accent is NOT used for: all borders, all hover states generically, decorative icons, section backgrounds, or body text.

### Contrast Verification

| Foreground | Background | Ratio | WCAG Level |
|------------|------------|-------|------------|
| `#ededed` (foreground) | `#0a0a0a` (background) | 17.4:1 | AAA |
| `#a1a1aa` (muted) | `#0a0a0a` (background) | 7.5:1 | AA |
| `#3b82f6` (accent) | `#0a0a0a` (background) | 4.6:1 | AA (large text) |
| `#60a5fa` (accent-hover) | `#0a0a0a` (background) | 6.7:1 | AA |
| `#ededed` (foreground) | `#141414` (surface) | 14.6:1 | AAA |
| `#ededed` (foreground) | `#3b82f6` (accent btn) | 3.8:1 | AA (large text, 14px+ bold) |

**Important:** Accent color at 4.6:1 passes AA for large text (18px+ or 14px+ bold) only. For body-size accent text, use `accent-hover` (#60a5fa) which passes AA at all sizes. Impact metrics are rendered at h3 size (24px) or larger, so #3b82f6 passes.

### Destructive Color

| Token | Hex | Usage |
|-------|-----|-------|
| Destructive | `#ef4444` | Not needed in Phase 1. Reserved for Phase 4 (form error states, validation messages) |

**Source:** D-01, D-02 (CONTEXT.md) locked the direction. RESEARCH.md provided specific hex values and contrast verification.

## Background Visual Depth (DSGN-05)

A subtle radial gradient emanating from the top-center of the page. Pure CSS, no image assets.

```css
body {
  background-color: var(--color-background);
  background-image: radial-gradient(
    ellipse 80% 50% at 50% -20%,
    rgba(59, 130, 246, 0.08),
    transparent
  );
  background-attachment: fixed;
}
```

**Rules:**
- Opacity must not exceed 0.10 -- current value 0.08 is the target
- Gradient color must match `--color-accent` RGB values (59, 130, 246)
- The gradient must be `fixed` so it does not scroll with content
- No animated gradient in Phase 1 (animations are Phase 5)
- No noise texture in Phase 1 (CSS gradient is sufficient; upgrade path exists if needed later)

**Source:** D-04 (CONTEXT.md) locked "subtle radial gradient or noise texture." RESEARCH.md recommended CSS gradient approach.

## Semantic HTML Contract (DSGN-04)

The root layout must establish these landmarks. Phase 1 renders them as empty containers where noted.

| Element | HTML | Role | Phase 1 State |
|---------|------|------|---------------|
| Banner | `<header role="banner">` | Site navigation | Empty placeholder (built in Phase 3) |
| Main content | `<main role="main">` | Primary content area | Contains placeholder page |
| Footer | `<footer role="contentinfo">` | Site footer | Empty placeholder (built in Phase 2) |

### Heading Hierarchy (Phase 1 placeholder page)

| Level | Content | Notes |
|-------|---------|-------|
| h1 | `{personalInfo.name}` | One per page, always |
| p (styled as subtitle) | `{personalInfo.title}` | NOT a heading -- styled text only |

**Accessibility baseline:**
- `<html lang="en">` set on root element
- Skip-to-content link: not required in Phase 1 (no navigation to skip past), add in Phase 3
- Focus indicators: browser defaults acceptable in Phase 1 (no interactive elements beyond two links)
- Color contrast: all text passes WCAG AA (verified in Color section above)

Using Tailwind v4 default breakpoints with mobile-first approach.

| Breakpoint | Width | Target Device |
|------------|-------|---------------|
| Default (no prefix) | 0px+ | Mobile (320px minimum) |
| `sm:` | 640px+ | Large phone / small tablet |
| `md:` | 768px+ | Tablet |
| `lg:` | 1024px+ | Small desktop |
| `xl:` | 1280px+ | Desktop |

### Phase 1 Responsive Rules

| Element | Mobile (< 768px) | Desktop (>= 768px) |
|---------|-------------------|---------------------|
| Page padding (horizontal) | 24px (`px-6`) | 48px (`px-12`) at md, 96px (`px-24`) at lg |
| Display text | 2.5rem | 4.5rem |
| Max content width | 100% | 896px (`max-w-4xl`) centered |
| CTA buttons | Full width, stacked | Inline, side by side (`flex gap-4`) |
