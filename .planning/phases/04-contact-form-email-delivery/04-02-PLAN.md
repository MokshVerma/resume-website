**State management:**
```typescript
type FormStatus = "idle" | "loading" | "success" | "error" | "rate-limited";

interface FieldErrors {
  name?: string[];
  email?: string[];
  message?: string[];
}
```

Use `useState` for:
- `status: FormStatus` (initial: `"idle"`)
- `fieldErrors: FieldErrors` (initial: `{}`)

**Form submission handler: `handleSubmit(e: FormEvent<HTMLFormElement>)`**
1. `e.preventDefault()`
2. Set `status` to `"loading"`, clear `fieldErrors`
3. Extract form data via `new FormData(e.currentTarget)`:
   - `name: formData.get("name") as string`
   - `email: formData.get("email") as string`
   - `message: formData.get("message") as string`
4. `try/catch` block:
   - `fetch("/api/contact", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ name, email, message }) })`
   - Parse response: `const data = await res.json()`
   - If `res.ok`: set status to `"success"`
   - Else if `res.status === 429`: set status to `"rate-limited"`
   - Else if `data.fieldErrors`: set `fieldErrors` to `data.fieldErrors`, set status to `"idle"`
   - Else: set status to `"error"`
   - Catch block: set status to `"error"`

**Reset handler: `handleReset()`**
- Set status to `"idle"`, clear fieldErrors
