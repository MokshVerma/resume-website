2. Create `src/app/api/contact/route.ts` as a POST Route Handler (NOT a Server Action -- per research recommendation, Route Handlers give direct access to request headers for IP-based rate limiting). The file must contain:

   **Imports:**
   - `import { type NextRequest } from "next/server"`
   - `import { Resend } from "resend"`
   - `import { contactSchema } from "@/lib/contact-schema"`

   **Resend client initialization:**
   - `const resend = new Resend(process.env.RESEND_API_KEY)`

   **Rate limit store (module-level):**
   - `const rateLimitMap = new Map<string, number>()`
   - `const RATE_LIMIT_WINDOW = 60_000` (1 minute in ms)

   **Helper: getClientIp(request: NextRequest): string**
   - Extract from `request.headers.get("x-forwarded-for")` -- take first value before comma, trim
   - Fallback to `request.headers.get("x-real-ip")`
   - Final fallback: `"unknown"`
   - NOTE: Do NOT use `request.ip` -- removed in Next.js 15

   **Helper: isRateLimited(ip: string): boolean**
   - Check `rateLimitMap.get(ip)` -- if exists and `Date.now() - timestamp < RATE_LIMIT_WINDOW`, return true
   - Otherwise return false

   **Helper: cleanupRateLimitMap(): void**
   - Iterate entries, delete any where `Date.now() - timestamp > RATE_LIMIT_WINDOW`
   - Called at start of each request to prevent unbounded growth

   **Helper: buildEmailHtml(name: string, email: string, message: string): string**
   - Returns an HTML string with:
     - Outer div: `font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif; max-width: 600px; margin: 0 auto; padding: 20px`
     - h2: "New Portfolio Contact" with `color: #333; border-bottom: 2px solid #3b82f6; padding-bottom: 10px`
     - Paragraphs for From (name), Email (mailto link), Sent (timestamp)
     - Timestamp formatted with: `new Date().toLocaleString("en-IN", { timeZone: "Asia/Kolkata", dateStyle: "medium", timeStyle: "short" })`
     - hr separator
     - Message div with `white-space: pre-wrap; line-height: 1.6; color: #374151`
   - Per D-09: simple HTML email, readable in any client

   **export async function POST(request: NextRequest)**
   Step 1 -- Cleanup + Rate limit:
   - Call `cleanupRateLimitMap()`
   - `const ip = getClientIp(request)`
   - If `isRateLimited(ip)`: return `Response.json({ error: "rate_limited", message: "You've already sent a message recently. Please try again in a minute." }, { status: 429 })`

   Step 2 -- Parse and validate:
   - `const body = await request.json()`
   - `const result = contactSchema.safeParse(body)`
   - If `!result.success`: return `Response.json({ error: "validation_error", fieldErrors: result.error.flatten().fieldErrors }, { status: 400 })`

   Step 3 -- Send email via Resend:
   - Destructure `{ name, email, message }` from `result.data`
   - Call `resend.emails.send()` with:
     - `from: "Portfolio Contact <onboarding@resend.dev>"` (use onboarding@resend.dev during dev, change to verified domain for production)
     - `to: "mokshverma98@gmail.com"`
     - `replyTo: email` (so Moksh can reply directly to the visitor)
     - `subject:` backtick template `Portfolio Contact: ${name}` (per D-10)
     - `html: buildEmailHtml(name, email, message)`
   - If `error`: return `Response.json({ error: "send_failed", message: "Something went wrong. Please try again." }, { status: 500 })`

   Step 4 -- Record + respond:
   - `rateLimitMap.set(ip, Date.now())`
   - Return `Response.json({ success: true })`

   Wrap the entire body parsing and Resend call in a try/catch. In the catch block, return `Response.json({ error: "send_failed", message: "Something went wrong. Please try again." }, { status: 500 })`.
